# MINI-Shop
小型商城项目实践
### 技术：
服务端：Spring+SpringMVC+MyBatis框架整合
数据库：MySql8.0
web服务器：Tomcat
项目管理：Maven
前端：jQuery+BootStrap+JavaScript
开发工具：idea2020
### 功能
实现登录功能。
实现登录成功后全部商品分页显示。
实现翻页功能。
添加监听器，实现获取全部商品类型功能。
实现商品增加功能。
实现商品删除功能。
实现商品批量删除功能。
实现商品修改的回显功能。
实现商品修改功能。
实现按商品名称条件查询并分页功能
实现按商品类型查询并分页功能
实现按商品价格查询并分页功能
实现按商品名称、商品类型、商品价格多条件查询并分页
实现退出功能。
